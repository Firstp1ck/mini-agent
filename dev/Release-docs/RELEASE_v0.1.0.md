# Release v0.1.0

## Overview

This is the first public release of **mini-agent**, a small desktop assistant you run on your own computer. You chat with **Claude (Anthropic)** or **GPT (OpenAI)**; the assistant can read and edit files in your project folder and run shell commands you approve by context—use it only where you are comfortable with that.

## Highlights

- **Simple first-time setup** — On first launch you pick your AI provider, enter your API key, choose a default model, and set how much “thinking” effort you want. Your choices are saved so the next run starts immediately.

- **Terminal or window** — Use the familiar command-line chat, or open the **graphical** chat window with the same capabilities.

- **Clear feedback** — While the model is replying or using tools, the GUI shows that something is happening so you are not left wondering.

- **Change settings anytime** — In the GUI you can open settings during a session to switch provider, model, or thinking level without hunting through config files by hand.

- **Your project, your rules** — If you add optional guide files (`AGENTS.md` or `CLAUDE.md`) in your project folder, mini-agent picks them up automatically so the assistant follows your team’s conventions.

Thank you for trying mini-agent. See the repository **README** for install steps, safety notes, and optional Windows executable build instructions.
